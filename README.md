#Disney Contest Boilerplate

####Technologies Used

  * NodeJS
  * NPM
  * Express
  * Webpack
  * Babel
  * React

####Getting Started In Development

In your terminal run the following:

```shell
npm install
npm run dev
```

Now in a new terminal tab run:

```shell
npm start
```

To build the JS out:

```shell
npm run build
```

Both the build process and server have watchers -- no need to reload!
